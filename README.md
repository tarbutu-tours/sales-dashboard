# 🚢 דשבורד מכירות – קרוזים מאורגנים

דשבורד מכירות חי המחובר ל-Pipedrive ו-Aircall.

## קבצים

| קובץ | תיאור |
|------|--------|
| `index.html` | הדשבורד הראשי |
| `codes.html` | ניהול קודי טיולים |

## חיבורים

- **Pipedrive** – לידים, עסקאות, משפך, מלאי מוצרים, נציגים
- **Aircall** – שיחות נכנסות / יוצאות / לא נענו
- **פרסום** – נתוני דמו (Google Ads + Facebook)

## שימוש

1. פתח את הקישור של GitHub Pages
2. הדשבורד מתחבר אוטומטית לנתונים
3. מתרענן כל 5 דקות
4. לעדכון קודי טיולים – לחץ על כפתור "⚙️ קודי טיולים"

## עדכון עתידי

לעדכון הדשבורד: ערוך את `index.html` ב-GitHub → הדשבורד מתעדכן אוטומטית.
